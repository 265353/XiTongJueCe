### 2.3 光储充优化数学模型速查

#### 目标函数: 全生命周期成本最小

```
min NPC = Σ(C_inv + C_om + C_rep + C_grid - C_sell - C_carbon)
         × (1+r)^(-t),  t = 1..T

其中:
C_inv  = 初始投资 (光伏+储能+充电桩+基础设施)
C_om   = 年运维费 (投资×运维率)
C_rep  = 更换成本 (逆变器/电池/充电桩)
C_grid = 购电费 (分时电价×购电量)
C_sell = 售电收入 (上网电价×上网电量)
C_carbon = 碳交易收入 (减排量×碳价)
r      = 折现率 (6-8%)
T      = 项目寿命 (20-25年)
```

#### 关键约束

```
1. 功率平衡: P_pv(t) + P_ess_dis(t) + P_grid(t) = P_charge(t) + P_bldg(t) + P_ess_ch(t)
2. SOC约束: SOC_min ≤ SOC(t) ≤ SOC_max  (0.10 ≤ SOC ≤ 0.90)
3. 储能功率: |P_ess(t)| ≤ P_ess_max
4. 变压器约束: P_grid(t) ≤ S_trans × cosφ  (2,500kVA × 0.95)
5. 可靠性: LOLP ≤ 0.02
6. 自洽率: RE_self_use ≥ 0.70
7. 面积约束: A_pv ≤ A_available
```

## 三、调度策略模板

### 3.1 规则式调度策略 (基线)

```python
# 时间尺度: 15min
def rule_based_dispatch(hour, pv_kw, charge_kw, bldg_kw, soc, ess_cap, ess_power):
    """
    基线调度策略: 光伏优先自用, 余电充储能, 不足时储能放电, 最后购电
    """
    net_load = charge_kw + bldg_kw - pv_kw  # 净负荷

    if net_load < 0:
        # 光伏过剩: 充电储能, 剩余上网
        charge_power = min(-net_load, ess_power, (0.90 - soc) * ess_cap / 0.25)
        grid_export = -net_load - charge_power
        grid_import = 0
    elif net_load > 0:
        # 光伏不足: 储能放电, 不足购电
        discharge_power = min(net_load, ess_power, (soc - 0.10) * ess_cap / 0.25)
        grid_import = net_load - discharge_power
        grid_export = 0
        charge_power = -discharge_power
    else:
        charge_power = 0
        grid_import = 0
        grid_export = 0

    return {
        'ess_power_kw': charge_power,  # 正=充电, 负=放电
        'grid_import_kw': grid_import,
        'grid_export_kw': grid_export,
    }
```

### 3.2 分时电价驱动策略 (经济最优)

```python
def tou_driven_dispatch(hour, season, pv_kw, charge_kw, bldg_kw, soc, ess_cap, ess_power, tou_price):
    """
    分时电价驱动: 谷时充电, 峰时放电, 光伏配合
    """
    price = get_tou_price(hour, season)
    net_load = charge_kw + bldg_kw - pv_kw

    if price <= 0.40:  # 谷价/深谷: 储能充电
        charge_power = min(ess_power, (0.90 - soc) * ess_cap / 0.25)
        grid_import = net_load + charge_power
        grid_export = 0
    elif price >= 1.20:  # 峰价/尖峰: 储能放电
        discharge_power = min(net_load, ess_power, (soc - 0.10) * ess_cap / 0.25)
        grid_import = max(0, net_load - discharge_power)
        grid_export = max(0, -net_load)
    else:  # 平段: 光伏自用优先
        if net_load < 0:
            charge_power = min(-net_load, ess_power)
            grid_export = -net_load - charge_power
            grid_import = 0
        else:
            discharge_power = min(net_load, ess_power)
            grid_import = net_load - discharge_power
            grid_export = 0

    return {
        'ess_power_kw': charge_power if price <= 0.40 else -discharge_power,
        'grid_import_kw': grid_import,
        'grid_export_kw': grid_export,
    }
```

## 四、性能评估指标体系

### 4.1 能源指标

| 指标 | 公式 | 基准值 | 优秀值 |
|------|------|:-----:|:-----:|
| 能源自洽率 | RE_gen / Total_consumption × 100% | 70-80% | >90% |
| 光伏自用率 | PV_self_use / PV_total × 100% | 60-75% | >85% |
| 弃光率 | PV_curtailed / PV_total × 100% | <5% | <1% |
| 储能利用率 | ESS_throughput / ESS_capacity × 365 | 50-65% | >75% |
| 电网依赖度 | Grid_import / Total_consumption × 100% | 20-30% | <10% |

### 4.2 经济指标

| 指标 | 公式 | 基准值 | 优秀值 |
|------|------|:-----:|:-----:|
| LCOE (光伏) | NPC_pv / ΣE_pv | 0.25-0.35元/kWh | <0.20元 |
| LCOE (储能) | NPC_ess / ΣE_ess_discharge | 0.40-0.60元/kWh | <0.35元 |
| 投资回收期 | Investment / Annual_net_income | 5-7年 | <5年 |
| IRR (内部收益率) | NPV=0时的折现率 | 10-15% | >18% |
| 度电利润 | (Revenue - Cost) / Total_kWh | 0.10-0.20元 | >0.25元 |

### 4.3 可靠性指标

| 指标 | 公式 | 基准值 | 优秀值 |
|------|------|:-----:|:-----:|
| LOLP (失负荷概率) | Hours_with_shortage / Total_hours | <2% | <0.5% |
| LPSP (缺电率) | Unmet_energy / Total_demand | <1% | <0.1% |
| 充电排队概率 | Queue_events / Total_arrivals | <5% | <1% |
| 平均等待时间 | ΣWait_time / N_arrivals | <10min | <3min |
