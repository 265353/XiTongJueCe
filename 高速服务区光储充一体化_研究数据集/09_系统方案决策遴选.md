# 系统方案决策遴选

## 一、综合评价指标体系

### 1.1 指标维度框架

```
                    ┌─────────────────────┐
                    │   目标层: 系统综合最优  │
                    └──────────┬──────────┘
                               │
        ┌──────────────────────┼──────────────────────┐
        │                      │                      │
  ┌─────┴─────┐          ┌─────┴─────┐          ┌─────┴─────┐
  │ 能源性指标  │          │ 经济性指标  │          │ 可靠性指标  │
  └─────┬─────┘          └─────┬─────┘          └─────┬─────┘
        │                      │                      │
   ┌────┼────┐            ┌────┼────┐            ┌────┼────┐
   │    │    │            │    │    │            │    │    │
   ▼    ▼    ▼            ▼    ▼    ▼            ▼    ▼    ▼
 新能源 可再生  自洽   全生命  运维  投资   缺电  供电  负荷
 自洽率 能源   率    周期成  成本  回收   概率  可靠  保障
       渗透率        本            期          性    率
```

### 1.2 具体评价指标

#### 能源性指标
| 指标 | 定义 | 计算公式 | 方向 |
|------|------|---------|:--:|
| 新能源自洽率 | 本地新能源满足负荷的比例 | E_self / E_total × 100% | ↑ |
| 可再生能源渗透率 | 可再生能源占总用能的比例 | E_renew / E_total × 100% | ↑ |
| 光伏消纳率 | 光伏被利用的比例 | E_pv_used / E_pv_gen × 100% | ↑ |
| 绿电自用率 | 绿电自用占绿电发电比例 | E_green_self / E_green_gen × 100% | ↑ |

#### 经济性指标
| 指标 | 定义 | 计算公式 | 方向 |
|------|------|---------|:--:|
| 全生命周期成本(NPC) | 折算到现值总成本 | Σ C_year / (1+i)^n | ↓ |
| 年运维成本 | 每年设备维护费用 | κ × C_inv | ↓ |
| 静态投资回收期 | 收回初始投资的时间 | C_inv / R_annual | ↓ |
| 内部收益率(IRR) | 使NPV=0的折现率 | — | ↑ |
| 平准化度电成本(LCOE) | 全生命周期单位电成本 | NPC / Σ E_annual | ↓ |

#### 可靠性指标
| 指标 | 定义 | 计算公式 | 方向 |
|------|------|---------|:--:|
| 缺电概率(LOLP) | 供电不足的概率 | N_shortage / N_total | ↓ |
| 供电可靠率 | 供电充足时间占比 | (1-LOLP) × 100% | ↑ |
| 1/2级负荷保障率 | 关键负荷绝不中断 | 必须达到100% | ↑ |
| 3级负荷缺电率限值 | 可中断负荷允许缺电 | ≤ 2% | ↓ |

#### 环境性指标（可选）
| 指标 | 定义 | 方向 |
|------|------|:--:|
| 年碳减排量 | CO₂减排量(吨/年) | ↑ |
| 绿电占比 | 绿电占总用电比例 | ↑ |

## 二、评价方法体系

### 2.1 方法总览

| 方法 | 类型 | 特点 | 适用场景 |
|------|:--:|------|---------|
| 层次分析法(AHP) | 主观赋权 | 依赖专家经验，1-9标度两两比较 | 准则层指标权重确定 |
| 熵权法 | 客观赋权 | 基于数据离散度，离散越大权重越大 | 指标层有实测数据时 |
| CRITIC法 | 客观赋权 | 综合考虑对比强度与冲突性 | 多指标综合评价 |
| TOPSIS法 | 综合评价 | 计算各方案与理想解的相对贴近度 | 多方案排序优选 |
| VIKOR法 | 综合评价 | 考虑群体效用最大化与个体遗憾最小化 | 折中方案选择 |
| 灰色关联度 | 综合评价 | 基于曲线几何相似性 | 小样本、信息不完全 |

### 2.2 组合赋权框架（推荐方案）

```
步骤1: 主观赋权 — 层次分析法(AHP)
  ├─ 构建层次结构模型（目标→准则→指标）
  ├─ 构造判断矩阵（两两比较，1-9标度）
  ├─ 计算权重向量（特征向量法）
  └─ 一致性检验（CR < 0.1）

步骤2: 客观赋权
  ├─ 熵权法 — 基于指标数据离散程度
  └─ CRITIC法 — 基于指标对比强度与冲突性

步骤3: 组合赋权
  ├─ 乘法合成: W_j = (W_ahp_j × W_entropy_j × W_critic_j) / Σ(...)
  └─ 或加法合成: W_j = α×W_ahp_j + β×W_entropy_j + γ×W_critic_j

步骤4: TOPSIS综合评价
  ├─ 构建加权标准化矩阵
  ├─ 确定正理想解(A+)和负理想解(A-)
  ├─ 计算各方案到正负理想解的欧氏距离
  └─ 计算相对贴近度 C_i = D_i- / (D_i+ + D_i-)

步骤5: 方案排序与遴选
  └─ 按C_i降序排列，C_i最大者为最优方案
```

## 三、AHP判断矩阵（参考标度）

### 3.1 1-9标度法

| 标度 | 含义 |
|:---:|------|
| 1 | 两因素同等重要 |
| 3 | 前者比后者稍微重要 |
| 5 | 前者比后者明显重要 |
| 7 | 前者比后者强烈重要 |
| 9 | 前者比后者极端重要 |
| 2,4,6,8 | 相邻判断的中间值 |
| 倒数 | 反比较（后者比前者重要） |

### 3.2 准则层判断矩阵（示例）

| | 能源性 | 经济性 | 可靠性 | 环境性 |
|---|:---:|:---:|:---:|:---:|
| **能源性** | 1 | 1/3 | 1/2 | 3 |
| **经济性** | 3 | 1 | 2 | 5 |
| **可靠性** | 2 | 1/2 | 1 | 4 |
| **环境性** | 1/3 | 1/5 | 1/4 | 1 |

> 此例中经济性权重最高，环境性权重最低。需根据实际项目调整。

### 3.3 一致性检验公式
```
CI = (λ_max - n) / (n - 1)
CR = CI / RI

其中RI(n=4) = 0.90, CR < 0.1 通过检验
```

## 五、敏感性分析

### 5.1 关键参数敏感性

| 参数 | 变化范围 | 对最优方案的影响 |
|------|:------:|------|
| 光伏组件价格 | ±30% | 大 → 价格下降时方案C可能反超 |
| 电价上涨率 | 0-5%/年 | 大 → 高上涨率时方案C更优 |
| EV渗透率增速 | ±50% | 中 → 影响充电负荷基础 |
| 储能电池价格 | ±40% | 大 → 价格下降时方案C/D受益 |
| 折现率 | 5%-10% | 中 → 高折现率不利于高投资方案 |

### 5.2 情景分析

| 情景 | 光资源 | 电价 | 政策补贴 | 推荐方案 |
|------|:----:|:---:|:------:|:------:|
| 乐观 | 好 | 高 | 有 | C(激进型) |
| 基准 | 中 | 中 | 无 | B(均衡型) |
| 悲观 | 差 | 低 | 无 | A(基础型) |
| 离网刚需 | 任意 | — | 有 | D(离网型) |

## 六、决策遴选流程

```
1. 明确决策目标与约束
   ├─ 投资预算上限
   ├─ 自洽率最低要求
   ├─ 场地面积限制
   └─ 政策合规要求

2. 生成备选方案集
   ├─ 光伏装机容量组合
   ├─ 储能配置方案
   ├─ 充电设施配置
   └─ 系统拓扑方案(交流/直流/混合)

3. 方案评估与筛选
   ├─ 技术可行性筛选（淘汰不可行方案）
   ├─ 经济可行性筛选（回收期<基准值）
   └─ 多维度综合评价（组合赋权+TOPSIS）

4. 敏感性分析
   ├─ 关键参数敏感性
   └─ 多情景鲁棒性验证

5. 最终方案推荐
   ├─ 基准情景最优方案
   └─ 不同情景下的备选方案
```
