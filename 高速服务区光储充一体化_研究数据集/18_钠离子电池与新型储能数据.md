## 一、钠离子电池 (2025-2026)

### 1.1 技术成熟度与量产进展

| 指标 | 数据 | 来源 |
|------|------|:----:|
| 2026年定位 | **规模化量产元年** | 行业共识 |
| 宁德时代量产时间 | **2026年内** | 官方公告 |
| 当前电芯价格 | **0.30-0.40元/Wh** | 市场调研 |
| 目标价格 | **<0.30元/Wh** | 宁德时代 |
| 远期价格 (2028+) | **~0.20元/Wh** | 伯恩斯坦 |

> **核心突破**：钠电成本已降至0.3元/Wh，接近LFP的0.28-0.35元/Wh，2026年底多家头部厂商批量交付储能及轻型动力钠电池。

### 1.2 钠离子电池技术参数

| 参数 | 钠离子电池 | LFP电池 (对比) | 优劣势 |
|------|:---------:|:----------:|:-----:|
| 电芯能量密度 | 140-180 Wh/kg | 160-200 Wh/kg | 钠电略低 |
| 系统能量密度 | 100-130 Wh/kg | 120-160 Wh/kg | 钠电低15-20% |
| 循环寿命 (80% DOD) | **4,000-8,000次** | 8,000-12,000次 | 钠电略低 |
| 日历寿命 | 10-15年 | 15-20年 | 钠电短 |
| 低温性能 (-20℃) | **88%容量保持率** | 70-80% | **钠电显著优** |
| 高温性能 (55℃) | 90%+ | 85-90% | 钠电略优 |
| 快充能力 | **15min充至80%** | 25-40min | 钠电优 |
| 安全性 (热失控) | **不起火不爆炸** | 需BMS管理 | 钠电更安全 |
| 原材料成本 | 碳酸钠 (便宜, 丰富) | 碳酸锂 (价格波动) | **钠电稳定** |
| 环保性 | 无毒, 可回收 | 含磷/重金属 | 钠电更优 |

### 1.3 钠离子电池在光储充场景的适用性分析

| 场景 | 适用性 | 原因 |
|------|:-----:|:----:|
| 北方寒冷地区服务区 | **★★★** | 低温性能优异, -20℃仍有88%容量 |
| 储能调峰 (非空间受限) | **★★★** | 成本低, 安全性好, 能量密度低可接受 |
| 光伏配套储能 | **★★☆** | 成本优势明显, 但需更大安装空间 |
| 超充功率缓冲 | **★★★** | 快充能力强, 适合高倍率场景 |
| 空间受限的屋顶储能 | ★☆☆ | 能量密度低, 重量大 |
| V2G车网互动 | ★☆☆ | 尚未有车规级量产产品 |

## 二、其他新型储能技术

| 参数 | 数据 |
|------|------|
| 系统效率 | 70-80% |
| 循环寿命 | **20,000+次** (理论上无限) |
| 日历寿命 | **25-30年** |

## 三、固态电池进展 (2025-2026)

| 指标 | 半固态 | 全固态 (预期) |
|------|:-----:|:---------:|
| 能量密度 | 300-400 Wh/kg | 400-500 Wh/kg |
| 安全性 | 显著提升 | 本质安全 |
| 量产时间 | **2025-2026 (车用)** | 2028-2030 |
| 成本 | 2-3x LFP | 远期1.5x LFP |
| 对光储充影响 | 间接(EV续航提升) | 间接(EV续航提升) |

> **影响分析**：固态电池主要提升EV续航(1000km+)，减少充电频次，间接影响服务区充电负荷。
> 2025年上汽MG4X搭载半固态电池上市，10万元级车型普及。
