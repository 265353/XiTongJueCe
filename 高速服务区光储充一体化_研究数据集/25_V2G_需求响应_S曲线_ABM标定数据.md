```

  可调节功率: 500kW × 50% = 250 kW (保留50% SOC应急)

  光伏限电: 峰值削减20% → 额外0.5~1万元
  充电桩功率调节: 高峰期降功率30% → 额外1~2万元
  合计DR年收入: 6~8万元 (中型服务区)
```

### 3.2 Bass扩散模型参数

Bass模型公式: $f(t) = \frac{p + q \cdot F(t)}{1 - F(t)}$, 其中 $F(t)$ 为累积渗透率

| 参数 | 符号 | 估计值 | 含义 |
|------|:----:|:------:|------|
| 创新系数 | p | 0.02 | 外部影响 (广告/政策) |
| 模仿系数 | q | 0.40 | 内部影响 (口碑/网络效应) |
| 承载容量 | K | 2.8~3.15亿辆 | 中国汽车保有量上限 |
| q/p比值 | — | 20 | >1说明口碑主导 |

**拟合方法**: 用2015-2025年保有量数据做非线性最小二乘拟合

```python
from scipy.optimize import curve_fit

def bass_model(t, p, q, K):
    """Bass扩散模型累积采用量"""
    return K * (1 - np.exp(-(p + q) * t)) / (1 + (q/p) * np.exp(-(p + q) * t))

# 历史数据 (2015=0, 2016=1, ...)
years = np.arange(2015, 2026) - 2015
ownership = np.array([42, 91, 153, 261, 381, 492, 784, 1310, 2041, 3140, 4397])  # 万辆

popt, pcov = curve_fit(bass_model, years, ownership, p0=[0.02, 0.40, 30000])
p_fit, q_fit, K_fit = popt
```

### 3.3 Logistic S曲线 (简化替代)

Logistic公式: $N(t) = \frac{K}{1 + e^{-r(t - t_0)}}$

| 参数 | 数值 | 含义 |
|------|------|------|
| 承载容量 K | 3.0亿辆 | NEV保有量上限 |
| 增长速率 r | 0.45 | 年化增长率参数 |
| 拐点 t₀ | 2024.5 | 50%渗透率时点 |

**拐点判断**: 2024年NEV月度销量渗透率已突破50%，保有量渗透率~17%，拐点已过S曲线中段

```python
LOGISTIC_K = 3.0e8                # NEV承载容量 (辆)
LOGISTIC_R = 0.45                 # 增长速率
LOGISTIC_T0 = 2024.5              # 拐点年份
EV_CHARGING_HIGHWAY_GROWTH = {    # 高速充电量年增长指数
    2020: 1.00, 2021: 2.28, 2022: 4.41, 2023: 8.11,
    2024: 10.88, 2025: 14.20, 2026: 17.5, 2027: 20.8,
    2028: 23.5, 2029: 25.8, 2030: 27.5,
}

# === capacity_optimization.py 修改 ===
def get_load_growth_factor(year, model='logistic'):
    """计算负荷增长因子"""
    if model == 'exponential':
        return (1.0 + LOAD_GROWTH_RATE) ** (year - BASE_YEAR)
    elif model == 'logistic':
        # S曲线: 基于NEV渗透率驱动充电负荷增长
        from scipy.special import expit
        penetration = expit(LOGISTIC_R * (year - LOGISTIC_T0))
        base_penetration = expit(LOGISTIC_R * (BASE_YEAR - LOGISTIC_T0))
        return penetration / base_penetration
```

### 4.5 ABM校准标准 (ASHRAE Guideline 14)

| 指标 | 公式 | 合格标准 | 优秀标准 |
|------|------|:-------:|:-------:|
| NMBE (月) | $\frac{\sum(S_i - M_i)}{\sum M_i} \times 100\%$ | ±5% | ±3% |
| CVRMSE (月) | $\frac{\sqrt{\frac{1}{n}\sum(S_i - M_i)^2}}{\bar{M}} \times 100\%$ | ≤15% | ≤10% |
| R² | 相关系数平方 | ≥0.85 | ≥0.95 |

> S=仿真值, M=实测值

```python
    """计算NMBE和CVRMSE"""
    n = len(measured)
```
