### 3.2 并网技术要求

| 参数 | 要求 | 标准来源 |
|------|------|:--------:|
| 功率因数 | ≥0.95 (超前/滞后) | GB/T 36547 |
| 电压偏差 | ±7% (10kV), ±10% (380V) | GB/T 12325 |
| 频率偏差 | ±0.5Hz (正常), ±1Hz (应急) | GB/T 15945 |
| 谐波畸变率 | THD<5% (电压) | GB/T 14549 |
| 防孤岛保护 | <2s动作 | GB/T 33593 |
| 低电压穿越 | 按标准曲线 | GB/T 19964 |

## 四、政策对经济性的影响量化

### 4.1 补贴对投资回收期的影响

| 项目 | 无补贴回收期 | 有补贴回收期 | 缩短幅度 |
|:----:|:---------:|:---------:|:-------:|
| 中型服务区(500kWp+500kWh) | 6.5年 | 5.0年 | -23% |
| 大型服务区(1MWp+1MWh) | 7.0年 | 5.5年 | -21% |
| 小型停车区(200kWp+200kWh) | 7.5年 | 6.0年 | -20% |

### 4.2 碳交易收入测算

| 参数 | 数值 |
|------|------|
| 1MWp光伏年减排 | ~800-1,000 tCO2 |
| CCER价格 | 70元/tCO2 |
| 年碳交易收入 | **5.6-7.0万元** |
| 1MWh储能年减排 (峰谷替代) | ~50-100 tCO2 |
| 储能碳交易收入 | 0.35-0.70万元/MWh/年 |

### 4.3 绿证收入测算

| 参数 | 数值 |
|------|------|
| 1MWp光伏年发电量 | ~100-150万kWh |
| GEC价格 | 30-50元/MWh |
| 年绿证收入 | **3.0-7.5万元** |
