### 1.2 四类太阳能资源区划

| 资源类别 | 年辐射量范围 (kWh/m²) | 代表地区 | 特点 |
|:------:|:----------------:|:--------:|:----:|
| **I类** (最丰富) | >1,750 | 西藏、青海、甘肃、宁夏、内蒙古西部 | 年日照>3,000h |
| **II类** (丰富) | 1,400-1,750 | 新疆、河北北部、山西北部、东北 | 年日照2,400-3,000h |
| **III类** (一般) | 1,050-1,400 | 华东、华中、华南大部分 | 年日照1,800-2,400h |
| **IV类** (较差) | <1,050 | 四川盆地、重庆、贵州 | 年日照<1,800h |

> **高速公路服务区适用性**：大部分高速公路位于II-III类资源区，光伏年等效利用小时数1,000-1,500h。
> IV类资源区（四川盆地）需适当增大光伏装机量或依赖储能调峰。

## 四、PVGIS/NSRDB数据获取指南

### 4.1 PVGIS API (欧盟, 免费, 全球覆盖)

| 参数 | 说明 |
|------|------|
| 空间分辨率 | ~1km (TMY) |
| 时间分辨率 | 1小时 (TMY) 或月均 |
| 数据范围 | 2005-2020 TMY |
| 覆盖中国 | ✅ (精度较好) |
| Python调用 | `pvlib.iotools.get_pvgis_hourly(lat, lon)` |
| Web界面 | https://re.jrc.ec.europa.eu/pvg_tools/ |

### 4.2 NSRDB (NREL, 免费, 全球覆盖)

| 参数 | 说明 |
|------|------|
| 空间分辨率 | 4km (PSM v3) |
| 时间分辨率 | 30分钟 / 1小时 |
| 数据范围 | 1998-2022 |
| 覆盖中国 | ✅ (精度中等) |
| Python调用 | `pvlib.iotools.get_psm3(api_key, lat, lon)` |
| API注册 | https://developer.nrel.gov/signup/ |

### 4.3 中国气象数据网 (国家站, 需申请)

| 参数 | 说明 |
|------|------|
| 空间分辨率 | 站点级 (~2,000站) |
| 时间分辨率 | 逐时 / 逐日 |
| 数据范围 | 1961-至今 |
| 申请方式 | http://data.cma.cn 注册, 科研用途免费 |
| 数据质量 | 高精度 (站点实测) |
