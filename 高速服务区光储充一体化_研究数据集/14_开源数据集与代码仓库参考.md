## 一、NBSDC 国家基础学科公共科学数据中心（★ 最直接相关）

国家重点研发计划"高速公路基础设施绿色能源自洽供给与高效利用系统关键技术研究"（2021YFB1600200）共发布了多个数据集：

### 1.1 高速公路与自洽能源系统运营保障综合数据集 ⭐

| 属性 | 内容 |
|------|------|
| **标识符** | `CSTR:16666.11.nbsdc.ytiZzq66` |
| **URL** | `https://nbsdc.cn/general/dataDetail?id=674b6906195d2661e1ba41a1&type=1` |
| **发布机构** | 华北电力大学 |
| **数据量** | 5.94 MB（6文件） |
| **格式** | `.mat`、`.xlsx` |
| **内容** | ① 自洽能源系统调度运行数据（光伏出力、负荷曲线、系统配置）；② **高速公路服务区电动汽车充电负荷数据**（蒙特卡洛模拟生成） |

**模型价值**：
- MATLAB `.mat` 格式可直接加载，含可再生能源出力、负荷曲线、电池充放电计划
- 包含正常/应急两种场景的调度策略
- 有蒙特卡洛生成的充电负荷数据

## 三、国际公开数据集

### 3.1 Caltech ACN-Data（EV充电）⭐

| 属性 | 内容 |
|------|------|
| **URL** | `https://ev.caltech.edu/dataset` |
| **规模** | **66,000-80,000条真实充电会话**（3个站点，2018-2020） |
| **API** | REST API + Python客户端(`acnportal`) |
| **字段** | 连接时间、断开时间、充完时间、kWh充电量、充电电流时序 |
| **论文** | Lee, Li & Low, "ACN-Data: Analysis and Applications of an Open EV Charging Dataset" (ACM e-Energy 2019) |
| **静态镜像** | `github.com/tongxin-li/ACN-Data-Static` |

**模型价值**：可直接用于充电行为建模（连接时间分布、充电电量分布），验证充电负荷预测模型

### 3.2 NREL EVI-Pro / Electrification Futures Study（充电负荷）

| 属性 | 内容 |
|------|------|
| **URL** | `https://data.nrel.gov` (搜索"EFS"或"EVI-Pro") |
| **内容** | 美国全国EV充电小时级负荷曲线（多种场景） |
| **格式** | CSV/JSON |
| **使用** | `github.com/schatzcenter/EVIPro-Fleet-Gen` 可直接调用生成 |

### 3.3 NSRDB / PVGIS / Renewables.ninja（太阳能资源）⭐

| 来源 | URL | 分辨率 | 覆盖 |
|------|-----|:---:|------|
| **NSRDB (NREL)** | `https://nsrdb.nrel.gov/` | 30min, 4km | 全球（含中国） |
| **PVGIS (EU)** | `https://re.jrc.ec.europa.eu/pvg_tools/` | 1h, TMY | 全球（含中国） |
| **Renewables.ninja** | `https://www.renewables.ninja/` | 1h, TMY | 全球（含中国） |
| **Solargis** | `https://solargis.com/` | 15min, 250m | 全球商业级 |

> ⚠️ 以上均可用 `pvlib-python` 的 `iotools` 模块通过API直接拉取数据：`pvlib.iotools.get_pvgis_hourly()`, `pvlib.iotools.get_psm3()` 等

## 五、数据下载操作指南

### NBSDC数据集下载步骤

```
1. 访问 https://www.nbsdc.cn 注册账号
2. 搜索"CSTR:16666.11.nbsdc.ytiZzq66"或关键词"高速公路自洽能源"
3. 进入数据详情页，点击"申请下载"
4. 填写用途说明（建议注明："用于高速公路服务区光储充一体化系统研究"）
5. 提交后等待审核（通常1-3个工作日）
6. 审核通过后下载 .mat / .xlsx 文件
```

### PVGIS API Python调用示例

```python
import pvlib
import pandas as pd

# 以西安（三类资源区）为例
lat, lon = 34.26, 108.94

# 获取TMY逐时数据
df = pvlib.iotools.get_pvgis_hourly(latitude=lat, longitude=lon,
    start=2020, end=2020,
    components=True,
    surface_tilt=30,    # 最佳倾角
    surface_azimuth=180 # 朝南
)[0]

# df 包含: G(i) (倾斜面辐照 W/m²), T2m (温度), Gb(i), Gd(i), G(h) (水平面)
# 直接用于光伏出力计算
```

### ACN-Data Python调用示例

```python
from acnportal import acndata

# 获取Caltech站点所有充电会话
sessions = acndata.get_sessions('caltech')

# 查看充电行为分布
sessions['kWhDelivered'].hist(bins=50)       # 充电量分布
sessions['connectionTime'].dt.hour.hist()     # 充电开始时间分布
sessions['duration'].hist(bins=100)           # 充电时长分布
```
