# 微网系统架构（微网结构）

## 一、系统总体架构

### 1.1 "源-网-荷-储"一体化架构

```
                        ┌──────────────────────────┐
                        │     大电网 (Utility Grid)  │
                        │       10kV / 35kV         │
                        └────────────┬─────────────┘
                                     │
                              ┌──────┴──────┐
                              │  箱式变压器   │
                              │ 800-1250kVA  │
                              └──────┬──────┘
                                     │
                        ┌────────────┴────────────┐
                        │    交流母线 AC Bus       │
                        │        400V             │
                        └────────────┬────────────┘
                                     │
        ┌──────────┬────────┬───────┼───────┬─────────┬──────────┐
        │          │        │       │       │         │          │
   ┌────┴────┐ ┌──┴──┐ ┌───┴───┐ ┌─┴───┐ ┌─┴───┐ ┌───┴───┐ ┌──┴──┐
   │ 光伏阵列 │ │逆变器│ │储能系统│ │充电桩│ │照明 │ │暖通空调│ │其他 │
   │PV Array │ │     │ │ BESS  │ │EVSE │ │     │ │ HVAC  │ │Load │
   └─────────┘ └─────┘ └───────┘ └─────┘ └─────┘ └───────┘ └─────┘
```

### 1.2 子系统组成

| 子系统 | 核心设备 | 功能 |
|--------|---------|------|
| **分布式光伏发电** | 光伏组件、逆变器、汇流箱 | 绿色电力生产 |
| **储能系统(BESS)** | 锂电池柜、PCS、BMS | 削峰填谷、平滑波动、备电 |
| **充电设施** | 直流快充桩、交流桩、充电堆 | 电动汽车充电服务 |
| **配电系统** | 箱变、并网柜、母排 | 电力变换与分配 |
| **能量管理平台(EMS)** | 监控系统、通信网络、云平台 | 运行监控、优化调度 |

## 二、系统拓扑架构方案

### 2.1 方案一：交流微电网（主流方案）

**特点**：
- 光伏经逆变器接入交流母线
- 储能通过PCS接入交流母线
- 充电桩直接接入交流母线
- 技术成熟，设备标准化程度高
- 适用于新建及改造项目

**优势**：技术成熟、设备成本低、施工简单
**劣势**：多级变换损耗（PV: DC→AC→DC 充电），综合效率降低3-5%

### 2.2 方案二：直流微电网（新型方案）

**特点**：
- 光伏、储能、充电桩均接入直流母线
- 仅通过1个AC/DC变换器与电网连接
- 光伏直接DC-DC供充电桩

**优势**：减少变换环节，综合效率提升3-5%
**劣势**：直流保护设备成本高，标准化程度低

**架构示意**：
```
  PV(DC) ──→ DC/DC ──┐
                      ├── DC Bus(750V) ──→ DC/DC ──→ 充电桩(DC)
  BESS(DC) ─→ DC/DC ─┘           │
                                  └──→ AC/DC ──→ 电网(AC)
                                            └──→ DC/AC ──→ 交流负荷
```

### 2.3 方案三：交直流混合微电网

**特点**：
- 同时具有交流母线和直流母线
- 通过双向AC/DC变换器互联
- 光伏和储能可灵活接入任一侧
- 兼具两种方案优势

### 2.4 方案四：多服务区链式微网

**特点**：
- 相邻服务区微网通过功率互联互济
- 高速公路呈带状分布，各子微网沿路分布
- 支持双向功率流动（in-city ↔ out-city）

**优势**（Wang et al., 2025）：
- 可再生能源利用率提升20.2%
- 自维持率提升6.4%
- 99.2%电网供电请求可通过日前购电满足

## 三、运行控制策略

### 3.1 并网运行模式

| 时段 | 策略 |
|------|------|
| 日间有光照 | PV→负荷(优先)→BESS充电(余电)→上网(仍有余) |
| 日间无光照(阴天) | BESS+电网联合供电 |
| 夜间低谷 | 电网供电+BESS充电(谷电) |
| 夜间高峰 | BESS放电(优先)+电网补充 |

### 3.2 离网运行模式

| 条件 | 策略 |
|------|------|
| 电网故障检测 | 自动切换孤岛模式 |
| 离网供电优先级 | 1级(消防/监控)→2级(基本照明)→3级(充电桩/空调) |
| 负荷控制 | 3级负荷可切除，1/2级必须保障 |
| 恢复并网 | 电网恢复后自动同期并网 |

### 3.3 经济优化调度

目标函数：
```
min Σ(C_grid × P_grid + C_ess_degradation × P_ess + C_curtailment × P_curtailed)
```

约束条件：
- 功率平衡约束
- 储能SOC约束 (10%-90%)
- 充放电功率约束
- 电网交互容量约束
- 光伏出力约束

## 四、能量管理系统（EMS）架构

### 4.1 功能分层

| 层级 | 功能 | 时间尺度 |
|------|------|:------:|
| 优化调度层 | 日前/日内优化调度计划 | 15min-24h |
| 实时控制层 | 秒级功率平衡与模式切换 | ms-s |
| 设备控制层 | 逆变器/PCS/充电桩底层控制 | μs-ms |
| 监测分析层 | 数据采集、统计、分析、报警 | — |

### 4.2 关键控制策略

- **光伏优先消纳**：光伏出力优先满足本地负荷
- **储能智能充放**：基于电价和SOC的优化充放电
- **负荷柔性调节**：充电桩功率可调、空调温度带宽
- **防逆流保护**：防止光伏电力倒送电网（部分场景要求）
