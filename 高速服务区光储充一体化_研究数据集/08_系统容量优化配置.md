# 系统容量优化配置

## 一、优化模型体系

### 1.1 主要优化方法对比

| 方法 | 目标函数 | 决策变量 | 求解算法 | 应用场景 |
|------|---------|---------|---------|---------|
| **两阶段鲁棒优化** | 日均总成本最小(min-max-min) | PV容量、储能容量/功率、充电桩数量 | NC&CG算法 | 考虑源荷不确定性 |
| **双层优化** | 年均等效成本最小 | 外层:容量配置 内层:运行调度 | 改进粒子群(IPSO) | 运行-配置联合优化 |
| **多目标优化** | 经济成本+用户满意度 | PV/储能/充电桩容量 | 多目标粒子群(MOPSO) | 含排队模型 |
| **NSGA-III** | 低碳+成本+自洽率 | 光伏/储能容量 | NSGA-III | 低碳场景多目标 |
| **遗传粒子群(EGPSO)** | 缺电概率最低/经济最优 | 风光储容量 | 含精英主义的GPSO | 离网微网 |

### 1.2 典型优化模型数学表述

**两阶段鲁棒优化模型**（中国电机工程学报, 2025）：

```
外层(min): 投资决策
  min { C_inv(x) + max min C_ope(x, u, y) }
   x                   u    y

  其中:
  x = [PV_cap, ESS_cap, ESS_power, N_piles]  — 容量配置变量
  u = [PV_output, EV_load]                     — 不确定性变量(最恶劣场景)
  y = [P_grid(t), P_ess_ch(t), P_ess_dis(t)]  — 运行调度变量

约束条件:
  - 功率平衡: P_pv + P_grid + P_ess_dis = P_load + P_ess_ch
  - 储能SOC: SOC_min ≤ SOC(t) ≤ SOC_max
  - 充电桩数量: N_min ≤ N_piles ≤ N_max
  - 场地约束: A_pv ≤ A_available
  - 新能源自洽率: η_self ≥ 90%
  - 缺电率: LOLP ≤ 0.02 (仅3级负荷)
```

**双层优化模型**（综合智慧能源, 2024）：

```
外层: 容量优化
  min C_total = C_inv_annual + C_ope_annual + C_grid_annual
  s.t. 场地约束、投资约束

内层: 运行优化(8760h年时序模拟)
  min C_ope = Σ[ C_grid(t) × P_grid(t) × Δt ]
  s.t. 功率平衡、设备约束、SOC约束
```

## 二、优化求解算法

### 2.1 算法特性对比

| 算法 | 收敛速度 | 全局寻优能力 | 适用问题 | 特点 |
|------|:------:|:---------:|---------|------|
| 改进粒子群(IPSO) | 中 | 强 | 连续变量优化 | 引入自适应惯性权重 |
| 遗传算法(GA) | 慢 | 强 | 离散/连续混合 | 全局搜索好但收敛慢 |
| EGPSO(精英GPSO) | 快 | 很强 | 多约束非线性 | 引入精英保留机制 |
| NC&CG(嵌套列约束生成) | — | 精确解 | 鲁棒优化 | 求解min-max-min问题 |
| NSGA-III | 中 | 强 | 多目标(≥3目标) | 基于参考点的高维多目标 |
| Gurobi/CPLEX | 快 | 精确 | MILP | 求解器，需线性化处理 |

### 2.2 遗传粒子群(EGPSO)算法要点

```
1. 初始化: 随机生成N个粒子(PV容量, 储能容量, 储能功率)
2. 评价: 计算每个粒子的适应度(总成本/缺电率)
3. 更新: 个体最优 + 全局最优 → 更新速度与位置
4. 精英保留: 每代保留最优的k个粒子直接进入下一代
5. 交叉变异: 精英粒子间交叉 + 随机变异
6. 终止: 达到最大迭代次数或收敛阈值
```

## 三、目标函数体系

### 3.1 经济性目标

| 成本项 | 计算公式 | 说明 |
|--------|---------|------|
| 光伏投资 | C_pv_inv = c_pv × P_pv_rated | 含组件、逆变器、支架、施工 |
| 储能投资 | C_ess_inv = c_ess_p × P_ess + c_ess_e × E_ess | 功率成本+容量成本 |
| 充电桩投资 | C_evse_inv = c_evse × N_piles | 含桩体、变压器扩容 |
| 运维成本 | C_om = k_om × C_inv | 通常为投资的1-3%/年 |
| 购电成本 | C_grid = Σ p(t) × P_grid(t) × Δt | 分时电价下的电网购电 |
| 设备更换 | C_rep | 储能电池10-15年更换 |
| 残值 | S_residual | 光伏25年、储能到期残值 |

### 3.2 常用经济评价指标

| 指标 | 公式 | 说明 |
|------|------|------|
| NPC(净现值成本) | Σ C_year / (1+i)^n | 全生命周期折现总成本 |
| LCOE(平准化度电成本) | NPC / Σ E_year | 单位发电成本 |
| IRR(内部收益率) | NPV=0时的折现率 | 项目盈利能力 |
| PBP(投资回收期) | C_inv / R_annual | 静态回收期 |
| ROI(投资回报率) | (R_annual - C_annual) / C_inv | 年化回报率 |

### 3.3 基准经济参数

| 参数 | 数值 |
|------|:---:|
| 光伏单位投资 | 3,000-4,500 元/kWp |
| 储能单位投资(功率) | 800-1,200 元/kW |
| 储能单位投资(容量) | 1,000-1,500 元/kWh |
| 直流快充桩 | 30,000-80,000 元/台 |
| 液冷超充 | 200,000-400,000 元/台 |
| 折现率 | 6%-8% |
| 项目评估周期 | 20-25年 |
| 储能更换周期 | 10-15年 |
| 光伏运维费率 | 1%-2%/年 |
| 储能运维费率 | 2%-3%/年 |
