# 服务区用能负荷特征

## 一、服务区能源消耗结构

### 1.1 能耗分项占比（无充电桩服务区）

| 能耗分项 | 占比 | 备注 |
|---------|:---:|------|
| 空调系统 | 36.82% | 最大耗能项，季节性波动大 |
| 照明装置 | 28.58% | 全年24h运行 |
| 生活热水 | 17.50% | 餐饮/住宿相关 |
| 动力设备 | 10.10% | 水泵、风机等 |
| 其他 | 7.00% | 办公、弱电等 |

### 1.2 含充电桩服务区能耗结构变化

| 能耗分项 | 原占比 | 含充电桩后占比 |
|---------|:---:|:-----------:|
| 充电桩 | — | 50%-73% |
| 空调系统 | 36.82% | 10%-18% |
| 照明装置 | 28.58% | 8%-14% |
| 其他 | 34.60% | 9%-17% |

> 充电桩是含充电设施服务区的绝对主导负荷

## 二、典型日负荷曲线特征

### 2.1 常规设施用电（不含充电桩）

| 时段 | 负荷水平 | 特征 |
|------|:------:|------|
| 23:00-次日6:00 | 最低 | 人员活动少，仅基础照明和待机负荷 |
| 7:00-8:00 | 局部峰值 | 洗漱、备餐、早餐时段 |
| 9:00-11:00 | 较低 | 工作日户外活动为主 |
| 11:00-14:00 | 中高峰 | 午餐备餐+空调负荷 |
| 14:00-17:00 | 较低 | 午后平稳期 |
| 18:00-22:00 | 最高峰 | 晚餐备餐+娱乐+照明+空调叠加 |

### 2.2 充电桩负荷叠加后

充电桩负荷双峰（10-13时、14-18时）与常规设施早晚峰叠加，形成：
- 上午综合峰：10:00-13:00（充电峰+午餐负荷）
- 晚间综合峰：18:00-22:00（常规设施峰+部分充电）

### 2.3 季节性差异

| 季节 | 相对负荷系数 | 主要影响因素 |
|------|:---------:|------------|
| 春季 | 0.85 | 温和气候，空调负荷低 |
| 夏季 | 1.25 | 制冷空调高峰，充电量略增 |
| 秋季 | 0.80 | 温和气候，最低负荷季节 |
| 冬季 | 1.40 | 供暖负荷大，电池活性下降充电量增加 |

## 四、用能负荷建模参数

### 4.1 主体建模(Agent-Based Modeling)参数

| 参数类别 | 数值/描述 |
|---------|---------|
| 驻守员工 | 37人(案例) |
| 住宿配置 | 21间单人宿舍+8间双人宿舍+20间标准间宾馆 |
| 宾馆入住率 | 70% |
| 工作日室内活动概率 | 0.2(9:00-15:00) |
| 节假日室内活动概率 | 0.6(9:00-15:00) |

### 4.2 多能源负荷特征

- **电力负荷**：照明、空调、充电桩、办公设备
- **热力负荷**：冬季供暖（热泵/电锅炉）、生活热水
- **冷负荷**：夏季制冷（中央空调/分体空调）
- **冷热电耦合**：可实现CCHP联供优化

## 六、数据来源

- 山东建筑大学：《高速公路服务区建筑能耗定额研究》（调研119个服务区）
- 华北电力大学：基于主体建模和改进FCM的服务区多能源负荷预测（2024）
- 长安大学：低碳视角下的秦岭高速公路服务区可再生能源协同供能设计
- 国家基础学科公共科学数据中心（NBSDC）：高速公路多源多态综合能源微网工程设计典型场景负荷数据集
- 《煤气与热力》2022年10月刊：高速公路服务区建筑能耗定额研究
