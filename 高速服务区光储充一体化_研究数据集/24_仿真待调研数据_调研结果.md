### 1.3 天气修正系数细化

**调研结论**: 基于Kasten-Czeplak公式和中国Angstrom-Prescott系数经验值，可提供实证修正。

**Kasten-Czeplak公式 (云量→辐照衰减)**:
```
GHI / GHI_clear = 1 - 0.75 × (N/8)^3.4

其中 N = 云量(0-8 oktas)
GHI_clear = 晴空辐照度 (W/m²)
```

**天气类型→云量→辐照衰减映射 (经验公式推导)**:

| 天气类型 | 云量范围 (oktas) | GHI/GHI_clear (Kasten-Czeplak) | 原config值 | 建议修正值 |
|:-------:|:-------------:|:--------------------------:|:---------:|:--------:|
| 晴天 (clear) | 0-1 | 1.00-0.99 | **1.00** | **1.00** |
| 晴转多云 (partly_cloudy) | 2-4 | 0.96-0.87 | **0.80** | **0.82** |
| 多云 (cloudy) | 5-6 | 0.77-0.63 | **0.55** | **0.58** |
| 阴天 (overcast) | 7-8 | 0.50-0.25 | **0.30** | **0.32** |
| 雨天 (rain) | 7-8 (降水) | 0.25-0.10 | **0.15** | **0.15** |

**Angstrom-Prescott系数 (中国实测值)**:
```
GHI/GHI_extraterrestrial = a + b × (n/N_max)

a, b值因地区而异 (中国范围):
  a = 0.117-0.345 (大气透明度)
  b = 0.345-0.703 (散射比例)

典型值 (华中地区):
  a = 0.23, b = 0.52
```

**实测验证数据**:
- 香河站(华北平原): 2005-2019年GHI增加趋势+1.32 W/m²/年
- 冬季散射占比高(60%+), 夏季直射占比高(70%+)
- 梅雨季(6-7月)华中持续阴雨15-30天, GHI降至晴天的20-30%

**代码集成建议**:
```python
# 细化 WEATHER_COEFF, 基于Kasten-Czeplak公式
WEATHER_COEFF = {
    'clear': 1.00,
    'partly_cloudy': 0.82,  # 原0.80, 上调
    'cloudy': 0.58,         # 原0.55, 上调
    'overcast': 0.32,       # 原0.30, 上调
    'rain': 0.15,           # 维持不变
}
# 或使用连续函数:
# weather_coeff = 1 - 0.75 * (cloud_oktas/8)**3.4
```

### 2.1 充电渗透率 (到达车辆≠充电车辆)

**调研结论**: 高速公路充电渗透率因车型和日类型差异显著。

**充电渗透率实测数据**:

| 车型 | 高速充电率 (占总车型) | 来源 |
|:----:|:----------------:|:----:|
| 纯电(BEV) | **67%** (进入服务区后充电) | 上海NEV 2026春节报告 |
| 插混(PHEV) | **14%** | 同上 |
| 增程(REEV) | **38%** | 同上 |
| 加权平均 (BEV 65%+PHEV 25%+REEV 10%) | **~50%** | 推算 |

> **关键区别**: "充电渗透率" = 进入服务区的EV中实际充电的比例，不等于高速上EV占总车流的比例。

**高速EV占总车流比例**:
- 2026五一: **24%** (1,540万NEV / ~6,400万总车辆)
- 工作日估计: 12-18% (基于2025年均渗透率)

**综合充电到达率 (占全部进站车辆)**:

| 日类型 | 高速EV占比 | 充电率 | 综合充电渗透率 | 说明 |
|:-----:|:--------:|:-----:|:----------:|:----:|
| 工作日 | 15% | 50% | **7.5%** | 日常通勤为主 |
| 周末 | 18% | 55% | **10%** | 短途出游 |
| 节假日 | 24% | 60% | **14.4%** | 长途出行, 充电刚需 |
| 春节 | 20% | 55% | **11%** | 返乡, 部分EV不跑高速 |

**时段分布 (充电高峰)**:
- 高峰: 10:00-17:00 (占日充电量60%+)
- 次高峰: 08:00-10:00, 17:00-19:00
- 低谷: 22:00-06:00 (<5%)

### 2.4 天气类型二阶Markov转移矩阵

**调研结论**: 缺乏中国气象局逐日天气类型统计的精确数据。提供基于气候学经验的推荐矩阵。

**一阶Markov转移矩阵 (华中地区, 年度平均, 5状态)**:

| 从\到 | 晴天 | 晴转多云 | 多云 | 阴天 | 雨天 |
|:----:|:---:|:------:|:---:|:---:|:---:|
| **晴天** | 0.55 | 0.25 | 0.12 | 0.05 | 0.03 |
| **晴转多云** | 0.25 | 0.35 | 0.25 | 0.10 | 0.05 |
| **多云** | 0.15 | 0.20 | 0.30 | 0.20 | 0.15 |
| **阴天** | 0.10 | 0.15 | 0.25 | 0.30 | 0.20 |
| **雨天** | 0.10 | 0.15 | 0.20 | 0.20 | 0.35 |

> **行和=1.0**, 对角线为持续概率(persistence)。
> 晴天持续概率最高(0.55), 雨天持续概率次高(0.35, 对应梅雨/连阴雨)。

**季节性修正**:

| 季节 | P(晴→晴) | P(雨→雨) | 特点 |
|:----:|:-------:|:-------:|:----:|
| 春季 | 0.50 | 0.40 | 天气多变, 梅雨前期 |
| 夏季 | 0.60 | 0.45 | 晴热持续, 梅雨6-7月 |
| 秋季 | 0.65 | 0.30 | 秋高气爽, 晴天持续 |
| 冬季 | 0.50 | 0.35 | 阴天持续, 偶有雨雪 |

**二阶Markov升级 (25×5矩阵)**:
二阶需要 `P(w_t | w_{t-1}, w_{t-2})`, 共25个条件分布。在缺少30年逐日数据的情况下, 推荐用以下近似:

```
P(w_t | w_{t-1}, w_{t-2}) ≈ α × P(w_t | w_{t-1}) + (1-α) × P(w_t | w_{t-2})

其中 α = 0.7 (一阶权重, 近期影响更大)
```

**代码集成建议**:
```python
# 升级 calendar_utils.py 的 generate_weather_markov()
WEATHER_TRANSITION_MATRIX = {
    'clear': {'clear': 0.55, 'partly_cloudy': 0.25, 'cloudy': 0.12, 'overcast': 0.05, 'rain': 0.03},
    'partly_cloudy': {'clear': 0.25, 'partly_cloudy': 0.35, 'cloudy': 0.25, 'overcast': 0.10, 'rain': 0.05},
    'cloudy': {'clear': 0.15, 'partly_cloudy': 0.20, 'cloudy': 0.30, 'overcast': 0.20, 'rain': 0.15},
    'overcast': {'clear': 0.10, 'partly_cloudy': 0.15, 'cloudy': 0.25, 'overcast': 0.30, 'rain': 0.20},
    'rain': {'clear': 0.10, 'partly_cloudy': 0.15, 'cloudy': 0.20, 'overcast': 0.20, 'rain': 0.35},
}

# 二阶近似:
SECOND_ORDER_ALPHA = 0.7  # 一阶权重
```

### 4.2 分布式光伏上网电价最新政策

**调研结论**: 2025年136号文是里程碑改革, 分布式光伏全面参与市场化交易, 固定上网电价取消。

**政策演变**:

| 时间 | 政策 | 上网电价 |
|:----:|:----:|:-------:|
| 2021前 | 固定补贴 | 0.85元/kWh (标杆) + 补贴 |
| 2021-2024 | 补贴退坡→平价 | 燃煤基准 (各省0.35-0.45) |
| 2025 | **136号文** | **市场化交易, 竞价上网** |
| 2026 | 机制电价过渡期 | 各省机制电价0.15-0.42 |

**136号文核心内容**:
- 所有可再生能源必须参与电力市场
- 固定上网电价(Feed-in Tariff)彻底取消
- 分布式光伏: 自发自用按零售电价计, 余电上网按市场价
- 现货市场部分省份出现**0.013元/kWh**极端低价

**各省机制电价 (2025-2026竞价结果)**:

| 省份 | 机制电价 (元/kWh) | 燃煤基准 (元/kWh) |
|:----:|:---------------:|:---------------:|
| 浙江 | **0.3816** | 0.4153 |
| 江苏 | 0.38-0.42 | 0.3910 |
| 湖北 | 0.35-0.40 | 0.4161 |
| 湖南 | 0.35-0.40 | 0.4500 |
| 安徽 | 0.35-0.38 | 0.3844 |
| 山东 | 市场化(无机制电价) | 0.3949 |
| 新疆 | **0.15** (地板价) | 0.2500 |

**对光储充系统的影响**:

| 场景 | 电价计算 | 典型值 |
|:----:|:-------:|:-----:|
| 自发自用 (光伏直接供充电/建筑) | 节省的零售电价 | **0.60-1.00元/kWh** |
| 余电上网 | 市场交易价/机制电价 | **0.15-0.42元/kWh** |
| 储能套利 | 峰谷价差 | **0.30-1.25元/kWh** |

> **关键变化**: 自发自用价值远大于上网价值。光储充系统应以最大化自用率为首要目标,
> 而非追求高上网量。当前config.py的FEED_IN_PRICE=0.35需区分省份, 并考虑市场化风险。

**代码集成建议**:

# 改为省份+政策模式

    'market_price_low': 0.15,      # 新疆等西部省份
    'mechanism_average': 0.35,     # 华中/华东机制电价均值
    'mechanism_high': 0.42,        # 江苏/浙江
