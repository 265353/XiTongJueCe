# 高速公路服务区光储充微网 — 设备级组网架构

## 一、微网拓扑结构

采用**交流微电网方案**（技术成熟，设备标准化），400V低压交流母线，10kV高压并网。

```
                         ┌──────────────────────────────┐
                         │      大电网 10kV/35kV         │
                         └──────────────┬───────────────┘
                                        │
                    ┌───────────────────┴───────────────────┐
                    │        10kV 进线柜 + 计量柜            │
                    │    (隔离开关 + 真空断路器 + 计量CT/PT)   │
                    └───────────────────┬───────────────────┘
                                        │
                         ┌──────────────┴──────────────┐
                         │   箱式变压器 2×1250 kVA     │
                         │   10kV/0.4kV, Dyn11, 6% Uk  │
                         └──────────────┬──────────────┘
                                        │
                         ┌──────────────┴──────────────┐
                         │   400V 交流母线 (铜排)       │
                         │   并网柜 (防孤岛+SPD+计量)    │
                         └──────────────┬──────────────┘
                                        │
     ┌──────────┬──────────┬───────────┼───────────┬──────────┬──────────┐
     │          │          │           │           │          │          │
  ┌──┴──┐   ┌──┴──┐   ┌───┴───┐   ┌───┴───┐   ┌───┴───┐  ┌───┴───┐  ┌──┴──┐
  │光伏  │   │储能  │   │充电桩 │   │建筑   │   │SVG/   │  │站用电 │  │备用  │
  │发电  │   │系统  │   │系统   │   │负荷   │   │无功   │  │系统   │  │接口  │
  │单元  │   │BESS  │   │EVSE  │   │       │   │补偿   │  │UPS    │  │      │
  └──┬──┘   └──┬──┘   └───┬───┘   └───────┘   └───────┘  └───────┘  └──────┘
     │          │           │
  ┌──┴──┐   ┌──┴──┐   ┌────┴─────┐
  │5台   │   │10台 │   │16台120kW │
  │225kW │   │215kWh│   │2台480kW │
  │逆变器│   │电池柜│   │液冷超充  │
  └─────┘   └─────┘   └──────────┘
```

## 四、关键响应时间链

```
电网故障 → 防孤岛检测(≤200ms) → 并网开关分闸(≤60ms) → 孤岛模式建立
                                          ↓
PV功率波动 → 逆变器调节(<100ms) → 直流母线稳定
                                          ↓
负荷突变   → PCS响应(<20ms) → 储能充放电补偿 → 频率/电压恢复
                                          ↓
BMS保护    → 过流检测(<30ms) → 短路保护(<10ms) → 电池解列
                                          ↓
EMS调度    → 优化计算(秒级) → 指令下发(亚秒级) → 设备执行
```
