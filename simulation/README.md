# 高速公路服务区光储充一体化系统仿真

## 项目概述

基于"高速公路服务区光储充一体化研究数据集"（14份数据文件），构建了从**蒙特卡洛充电负荷预测 → 光伏出力合成 → 容量优化配置 → 多方案评价**的完整仿真链。仿真粒度到设备级（组件/逆变器/PCS/充电桩/变压器），生成7张论文级图表和量化分析结果。

## 仿真流程 (6步)

### 1. 蒙特卡洛充电负荷仿真 (`mc_charging_load.py`)

基于概率抽样模拟每辆EV的充电行为，含**排队模型**：

| 参数 | 分布 | 来源 |
|------|------|------|
| 日行驶里程 | LN(3.2, 0.88²) | 文件05 |
| 起始SOC | Gamma(2.8, 0.075) | 文件02/11 |
| 百公里电耗 | N(车型相关, σ²) | 文件02 |
| 电池容量 | N(车型相关, σ²) | 文件11 (7种车型) |
| 充电起始时间 | GMM(k=2, μ1=11, μ2=15) | 文件11 |
| 充电功率 | 离散{60:0.3, 120:0.5, 240:0.2} | 文件03 |

- 车辆到达：Poisson过程，小时级车流量来自文件11
- **排队模型**: 26个终端, 等待超时弃充（工作日45min → 春运15min）
- 每次仿真重复 N=5,000 次，输出 P5/P25/P50/P75/P95 置信区间
- 覆盖4种日类型：工作日/双休日/节假日/春运

### 2. 光伏出力合成 (`pv_generation.py`)

- 4季 × 24h逐时归一化系数（文件10）
- 5种天气类型（晴/晴转多云/多云/阴/雨）× 月度分布
- 生成8760h年度出力序列

### 3. 容量优化 (`capacity_optimization.py`)

粒子群算法（PSO）求解最优光伏+储能配置：

```
决策变量：P_pv (kWp), E_ess (kWh), P_ess (kW)
目标函数：min NPC = 投资 + Σ(运维+网购电)/(1+r)^n + 更换 - 残值
约束条件：
  - TOU感知最优调度 (24h迭代套利)
  - SOC: 10% ≤ SOC ≤ 90%
  - 自洽率 ≥ 70% (软约束, 带惩罚)
  - 变压器容量 ≤ 2,500kVA × 0.95
  - 场地面积 ≤ 可用面积
  - 电池逐年衰减 (循环+日历)
```

**调度策略**: 24h LP迭代套利 — ESS在谷电价/弃光时段充电, 峰电价时段放电。
**电网成本**: 使用季节性分时电价逐时计算, 非均价。
**验证**: PSO用典型日加权(快), 优化后用8760h全时序终验。

### 4-6. Pareto前沿 / 敏感性分析 / 多方案对比

非支配排序探索NPC vs 自洽率 trade-off，光伏成本±40%×电价±40%热力图，4方案雷达图。

## 微网架构概要

```
大电网10kV → 2×1250kVA箱变 → 400V交流母线
  ├── 光伏: 1231kWp (TOPCon 620Wp×1986 + 225kW逆变器×5)
  ├── 储能: 约2000kWh (LFP 215kWh/柜 + 100kW PCS)
  ├── 充电: 16×120kW快充 + 2×480kW液冷超充堆 (26终端)
  ├── 建筑负荷: 峰值~150kW
  ├── SVG: 50kVAR (最恶劣工况: 夜间纯建筑负荷 PF=0.85)
  └── 站用电/UPS
```

详见 `microgrid_architecture.md`
